# AI Agent 첫 번째 과제

## 설명
사용자의 질문에 따라 적절한 함수를 선택하고, 함수명과 인자(arguments)를 올바르게 출력하는 에이전트를 구현했습니다.

---

## 요구사항
- 모델은 gpt-4o-mini를 사용
- LLM에게 함수 설명 프롬프트 작성
- 함수명과 인자를 올바르게 출력하는지 확인

---

## 사용 모델
- gpt-4o-mini

---

## 실행 결과

### 질문 1
질문: 지금 인기 있는 영화가 무엇인지 알려줘  
모델 답: get_popular_movies()

### 질문 2
질문: movie ID 550에 해당하는 영화가 무엇인지 알려줘  
모델 답: get_movie_details(550)

### 질문 3
질문: movie ID 550에 해당하는 영화에 누가 출연하는지 알려줘  
모델 답: get_movie_credits(550)


# AI Agent 두 번째 과제

## 설명
사용자의 취향과 이미 본 영화를 기억하여 개인화된 영화 추천을 제공하는 챗봇을 구현했습니다.

---

## 요구사항
- 대화 기록을 저장하기 위한 messages 리스트를 구현하세요.
- 사용자 메시지(role: "user")와 AI 응답(role: "assistant")을 모두 append하세요.
- 이미 본 영화는 추천하지 않도록 메모리를 활용해야 합니다.

---

## 사용 모델
- gpt-4o-mini

---

## 실행 결과

User: 난 공포영화를 좋아해
AI: 공포영화는 정말 흥미로운 장르죠! 어떤 유형의 공포영화를 좋아하시나요? 심리적 공포, 초자연적 공포, 좀비 영화 등 다양한 스타일이 있는데, 말씀해 주시면 추천해 드릴 수 있어요!
User: 전설의고향을 봤고 여고괴담을 봤어
AI: 두 영화 모두 한국 공포영화의 대표작들이죠! **전설의 고향**은 전통적인 한국의 민속 이야기를 바탕으로 한 호러 앤 솔로 뽑힐 수 있는 작품이고, **여고괴담** 시리즈는 젊은 세대에게 큰 인기를 끌었던 공포 영화입니다. 

이 외에도 추천하고 싶은 한국 공포영화가 몇 가지 있어요:

1. **곤지암** - 실제 정신병원을 배경으로 한 공포영화로, 많은 긴장감을 주는 작품입니다.
2. **사라진 사람** - 실종 사건을 다룬 심리적 스릴러로, 긴장감이 넘치는 스토리입니다.
3. **부산행** - 좀비 영화이면서도 가족 간의 관계와 인류애를 다룬 감동적인 요소가 있는 작품입니다.

더 보고 싶은 종류나 스타일이 있으시다면 말씀해 주세요!
User: 내 정보를 바탕으로 추천 해줄 영화 있어?
AI: 물론입니다! 귀하가 **전설의 고향**과 **여고괴담**을 좋아하신다면, 다음과 같은 영화들을 추천드립니다:

1. **여고괴담 2: 생존** - 이 영화는 여고괴담 시리즈의 두 번째 편으로, 학생들 간의 갈등과 초자연적 사건이 결합된 이야기입니다.

2. **무서운 이야기** - 여러 개의 단편 공포 이야기가 결합된 앤솔러지 영화로, 각기 다른 방식으로 긴장감을 주는 작품입니다.

3. **마스터즈 오브 호러** - 여러 감독들의 단편 공포영화를 모은 시리즈로, 다양한 스타일의 공포를 느낄 수 있습니다.

4. **악녀** - 스릴러와 액션 요소가 결합된 영화로, 강렬한 여성 주인공과 함께 비극적인 배경 이야기가 담겨 있습니다. 공포보다는 스릴을 더 강하게 느낄 수 있습니다.

5. **기담** - 여러 개의 민속 전설을 바탕으로 한 현대적인 공포 영화로, 한국 전통 문화의 공포를 느낄 수 있습니다.

더 궁금한 점이나 다른 스타일의 영화를 원하신다면 말씀해 주세요!
