# AI Agent 첫 번째 과제

## 설명
사용자의 질문에 따라 적절한 함수를 선택하고, 함수명과 인자(arguments)를 올바르게 출력하는 에이전트를 구현했습니다.

---

## 요구사항
- 모델은 gpt-4o-mini를 사용
- LLM에게 함수 설명 프롬프트 작성
- 함수명과 인자를 올바르게 출력하는지 확인

---

## 사용 모델
- gpt-4o-mini

---

## 실행 결과

### 질문 1
질문: 지금 인기 있는 영화가 무엇인지 알려줘  
모델 답: get_popular_movies()

### 질문 2
질문: movie ID 550에 해당하는 영화가 무엇인지 알려줘  
모델 답: get_movie_details(550)

### 질문 3
질문: movie ID 550에 해당하는 영화에 누가 출연하는지 알려줘  
모델 답: get_movie_credits(550)
